# Notice

FlowTrim is inspired by the token-saving use cases represented by RTK, Ponytail, and Headroom:

- RTK-style command-output reduction
- Ponytail-style pressure toward simpler code generation
- Headroom-style direct compression for auditable long context

FlowTrim does not vendor those projects, copy their benchmark claims, or require them to be installed. Optional integrations must preserve upstream attribution and license terms before public release.
